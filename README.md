## Start Server Locally

```
npm i -g pnpm

pnpm i
pnpm run dev
```

And then visit `http://localhost:3000`

Or install the [Vitex extension for VS Code](https://marketplace.visualstudio.com/items?itemName=nyxb.vitex) to edit side-by-side.
